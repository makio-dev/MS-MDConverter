# プロジェクト概要

このドキュメントはMarkdownからExcel/Wordへの変換テスト用です。

## 機能一覧

| 機能名 | 説明 | ステータス |
| --- | --- | --- |
| MD解析 | Markdownを<br>ブロック単位でパース | 完了 |
| Excel出力 | 見出し・表・リストを<br/>Excelに変換 | 完了 |
| Word出力 | 見出し・表・リストを<br />Wordに変換 | 完了 |

## 対応要素

- 見出し (h1〜h6)
- テーブル（表）
- 箇条書きリスト
- コードブロック
- 段落テキスト
- 水平線

### コードブロックの例

```python
def hello():
    print("Hello, World!")
```

---

## 補足

**太字**や*斜体*、`インラインコード`、[リンク](https://example.com)はプレーンテキストに変換されます。

1. 手順1: ファイルを準備する
2. 手順2: スクリプトを実行する
3. 手順3: 出力ファイルを確認する
